# Streamlit Sitemap URL Extractor

A simple web application built with Streamlit to extract URLs from XML sitemaps.

## Features

- Extract URLs from XML sitemaps
- Parse sitemaps online
- Generate and download a CSV file of extracted URLs

## Installation

1. Clone this repository:
   ```
   git clone https://github.com/yourusername/streamlit-sitemap-url-extractor.git
   cd streamlit-sitemap-url-extractor
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the Streamlit app:
   ```
   streamlit run st-sitemap.py
   ```

2. Open your web browser and go to the URL displayed in the terminal (usually `http://localhost:8501`).

3. Enter the URL of the sitemap you want to parse in the input field.

4. Click the "Fetch URLs" button.

5. View the extracted URLs and download the CSV file using the "Download CSV" button.

## Requirements

- Python 3.6+
- Streamlit
- Requests
